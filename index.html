使用说明

注意： 请确保在使用本程序之前已安装必要的依赖项。

安装步骤

  1. 确保您已安装 Python 3.x。
  2. 下载项目文件。
  3. 在项目根目录下打开命令行。
  4. 运行以下命令安装依赖项： 双击打开install_libs.bat

运行程序

在命令行中输入以下命令启动程序：

python main.py
  • 也可以双击执行EXE可执行程序

功能介绍

  • 系统监控：实时监控 CPU 和内存使用率。
  • 邮件通知：在资源使用超出设定阈值时发送警报邮件。

常见问题

如果在使用过程中遇到问题，请检查以下内容：

  • 确保依赖项已正确安装。
  • 检查 Python 版本是否符合要求。
  • 确认配置文件路径是否正确。
